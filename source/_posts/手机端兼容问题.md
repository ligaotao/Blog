# 手机端兼容问题

> 记录开发中遇到的手机兼容问题

1. 在`iphone`中初始化时间与`其他浏览器不一致` ，

   ```javascript
   let date = '1992-02-05'
   let newDate = new Date(date)
   // 在iphone 中会得到 NaN
   // 在其他浏览器能得到预期的值 Sun Feb 05  192 00:00:00 GMT+0800 (中国标准时间)
   // 解决办法 将 - 替换为 /
   data.replace(/-/g,"/")
   ```
